# Pontos de Aruanda 🌟

Uma aplicação web premium para busca e visualização de pontos (letras) de Umbanda, com um design místico e interativo.

![Banner](https://images.unsplash.com/photo-1519751138087-5bf79df62d5b?auto=format&fit=crop&q=80&w=1000)

## ✨ Funcionalidades

- **Busca por Linhagem**: Filtre entre 10 linhagens principais (Caboclo, Exu, Preto-Velho, etc.).
- **Quantidade Personalizada**: Escolha ver de 1 a 10 pontos por vez.
- **Entidade Específica**: Busque por nomes específicos (ex: Zé Pelintra, Maria Padilha).
- **Simulação de Busca**: Animação imersiva que simula uma pesquisa em tempo real na internet.
- **Design Premium**: Interface com estrelas animadas, efeito de vidro (glassmorphism) e cores vibrantes.
- **Copiar Letra**: Botão para copiar rapidamente a letra para a área de transferência.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e SEO.
- **CSS3**: Estilização avançada, animações e layout responsivo.
- **JavaScript (Vanilla)**: Lógica de busca, manipulação do DOM e banco de dados local.

## 🚀 Como Executar

Não é necessário instalar nada! Basta baixar os arquivos e abrir o `index.html` em qualquer navegador moderno.

1. Baixe o código fonte.
2. Abra a pasta do projeto.
3. Clique duas vezes no arquivo `index.html`.
